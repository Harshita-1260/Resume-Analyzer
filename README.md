## .....RESUME ANALYZER....

## 🌐 Live Demo:resume-analyzer-bugerw2dbpiktr2jizhsvn.streamlit.app

## ✨ Features

- 📄 Upload resumes in PDF format.
- 📝 Enter a job description for comparison.
- 🤖 Calculates resume-job similarity using Machine Learning.
- 📊 Displays a resume match score with graphical visualization.
- 🔍 Extracts text automatically from PDF resumes.
- ⚡ Simple and interactive Streamlit web interface.
- 🎯 Helps optimize resumes for better job matching.

- ## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
- PyPDF2
- NLTK
- Matplotlib
- NumPy
- Pandas

- ## 🚀 Project Workflow

1. Upload a PDF resume.
2. Enter the target job description.
3. Extract text from the uploaded resume.
4. Preprocess the text using NLP techniques.
5. Convert text into numerical vectors using TF-IDF.
6. Calculate similarity using Cosine Similarity.
7. Display the resume match score and visualization.
